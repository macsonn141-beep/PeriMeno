# PeriMeno

PeriMeno is an offline-first SwiftUI iOS app scaffold for perimenopause and menopause symptom tracking.

The MVP focuses on fast local logging, simple pattern visibility, doctor-ready report preparation, privacy-first settings, premium gating scaffolds, and localization readiness.

## Principles

- No account
- No server
- No ads
- No third-party analytics
- Local SwiftData persistence
- Optional Health access
- Optional biometric lock

## Build

Open `PeriMeno.xcodeproj` in Xcode and build the `PeriMeno` target.
