# Launch Checklist

## Build And Signing

- [ ] Set the real Apple Developer Team in Xcode.
- [ ] Confirm production bundle identifier.
- [ ] Confirm version and build number.
- [ ] Archive a Release build.
- [ ] Upload to App Store Connect.
- [ ] Test the App Store Connect/TestFlight build on a physical device.

## App Store Assets

- [ ] Confirm final app icon.
- [ ] Capture required iPhone screenshots.
- [ ] Capture required iPad screenshots or opt out if iPad is not supported.
- [ ] Prepare app preview video only if useful.
- [ ] Finalize subtitle, description, keywords, and promotional text.

## Legal And Privacy

- [ ] Publish privacy policy URL.
- [ ] Publish support URL.
- [ ] Publish marketing URL if used.
- [ ] Complete App Privacy nutrition labels.
- [ ] Confirm privacy manifest matches actual app behavior.
- [ ] Confirm medical disclaimer appears in app and reports.
- [ ] Review health-related claims for App Review risk.

## Premium And StoreKit

- [ ] Create App Store Connect products for monthly, yearly, and lifetime.
- [ ] Confirm product IDs match the app code.
- [ ] Configure subscription group and localizations.
- [ ] Test purchase, restore, cancel, and expired subscription states in sandbox.
- [ ] Confirm premium-gated features are usable after purchase.

## QA

- [ ] Run onboarding, log, Brain Fog Mode, Home, Insights, Reports, Settings, History, edit, and delete flows.
- [ ] Test low-data and empty-data states.
- [ ] Test PDF generation and share/export.
- [ ] Test Health access denial and unavailable-device states.
- [ ] Test Arabic right-to-left layout after translation polish.
- [ ] Verify no placeholder URLs, emails, screenshots, or metadata remain.
